## Agent Logs

* Fixed "move across board wrap" issue by adding `IgnorePointer` to the Grid lines, Stars, and Fog of War overlays in `GameBoard`, which were previously blocking taps to the grid squares underneath.
* Implemented the "Re-tether" functionality for ships:
    * Added `isRetethering` state to `gameplayUiProvider`.
    * Updated `PlanningPanel` to toggle re-tethering mode for ships requiring a tether.
    * Enhanced `GameBoard` to highlight valid target Star Cities (friendly, anchored, within range, and with capacity) with a white border when re-tethering.
    * Implemented `TetherAction` creation in `_handlePieceTap` during re-tethering mode.
* Refined the Bombardment targeting UI:
    * Created `TargetIcon` widget (`target_icon.dart`) with a custom SVG crosshair design for consistent sizing.
    * Integrated `TargetIcon` into `GameBoard` to represent planned bombardments on target pieces.
    * Removed manual crosshair drawing from `BombardPainter`, which now only draws the dashed connection lines.
    * Removed the "Cancel" `[x]` button from `PlanningPanel` and kept the `gps_fixed` crosshair icon active during targeting mode.
    * Ensured that targeting mode is cancelled when tapping an illegal square or deselecting the piece.
* Implemented "Deselect to cancel" logic:
    * Modified `_handlePieceTap` in `GameBoard` to remove all pending actions (moves, bombards, placements) for a piece when it is deselected.
    * Updated piece selection logic to allow selecting pieces that were just placed, specifically to enable deselecting them to cancel their placement.
* Verified all changes with `flutter analyze`.
